# Automated-Video-Management-System
A list of tool for the youtuber to mangement their channel automatically 

There have 2 folder in github : 

PodCard Channel : Scrapping everyStory with it's url ( Support with https://www.tieuhoadan.net/)
- StoryScrappingTool : It is a tool can scrapping all the data form a story just put the menu url
+ It will store the story in folder will create automatically with name as name of Story
+ There have 2 file.txt : Data.txt ( have all the detail of story ) and Describle.txt ( the hook of story)


- VoiceTransference : Convert text to voice ( Support with https://vbee.vn/) 
+ The speed will be set at 1.1 and default voicer 
+ After download they will be store inside folder as StoryName

- StoryGoogleUpload : Up to the drive and Sheet ( Using Google API)
Notice : You must you your own API Key and credential.json

The Sheet will list auto with format : Number | Name | LinkDrive | Status
